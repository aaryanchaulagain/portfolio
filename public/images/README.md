# Image assets

Replace these placeholders with your own optimized images before launch.

## Required files

| Path | Purpose |
|------|---------|
| `profile.jpg` | Hero portrait (`[PROFILE_IMAGE_PATH]`) |
| `about-work.jpg` | About section working photograph |
| `og-default.jpg` | Open Graph / social share image (1200×630) |
| `awards/[HACKATHON_IMAGE_2026]` | 2026 hackathon / award visual |
| `awards/[AWARD_IMAGE_2025]` | 2025 award visual |
| `awards/[CERTIFICATE_IMAGE]` | Certificate image |
| `projects/*.jpg` | Project cover images |

## Guidelines

- Prefer WebP or compressed JPEG
- Portrait: roughly 800×1000 or similar
- Project cards: 1600×1000 landscape
- Do not commit confidential client screenshots
- Keep alt text meaningful in components
